## 💡 Tips for Best Results

### 1. Start Simple

- Begin with just a **Title** and **Concept**
- Let the AI handle Genre, Mood, and Structure
- Fine-tune only after seeing the initial results

### 2. Use the Analyzer

- Paste reference lyrics to quickly configure settings
- Describe your ideal sound in natural language
- Works well combined with manual tweaking afterwards

### 3. Structure Matters

- Genre-specific structure presets are optimized for that style
- Custom structures give you complete control
- Add chord progressions and performance notes per section for maximum precision

### 4. Iterate Sections

- Don't regenerate the whole song if only one section isn't working
- Use the **↻ Regen** button on individual sections
- All other sections are preserved

### 5. Save Frequently

- Use the **History** tab to save versions as you work
- Export to `.txt` files for external backups
- Previously exported songs can be re-imported at any time

### 6. Vocal Configuration

- For specific vocal styles, configure the **Vocal** tab details
- Leave fields as "AI Choose" for automatic selection
- **Instrumental** mode provides performance directions instead of lyrics

### 7. Sound Layering

Combine multiple Sound tab options for a rich sonic palette:

| Field | Best Practice |
| --- | --- |
| **Instruments** | Use for specific instrument selection (e.g. "Moog Synthesizer, Fretless Bass, Timpani") |
| **Instrumentation Focus** | Use for general approach (e.g. "Acoustic", "Electronic") |
| **Bass** | Controls bass character independently of overall instrumentation |
| **Spatial/Effects** | Specify multiple effects simultaneously |
| **Mix Character** | Multi-select for nuanced production direction |
| **Influences** | Keep to 1–3 artists maximum for best results |

### 8. Granular Control

- **Goal** and **Groove Feel** help the AI understand the song's purpose and pocket
- **Rhythm** styles (Triplet, Swing, Syncopated) influence lyric phrasing and flow
- Leave settings unselected or set to "AI Choose" to let the AI decide — it uses genre and mood as context
- Custom text inputs are available for most options when presets don't fit

## 🔄 Common Workflows

### Workflow 1: Quick Generation

Best for: Getting started fast with a new idea.

1. **Settings Tab** — Enter a title and concept
2. Click **Write My Song**
3. **Export** to Suno

### Workflow 2: Style-Based Generation

Best for: Recreating a specific sonic aesthetic.

1. Click **Analyze**
2. Describe your desired sound in natural language
3. Review the auto-configured settings
4. Generate

### Workflow 3: Lyrics-First

Best for: When you have partial lyrics or a lyrical style in mind.

1. **Lyrics Tab** — Paste your lyrics
2. Choose an AI mode (Complete / Rewrite / Match / Fit)
3. Optionally use **Analyze** to extract structure from your lyrics
4. Generate

### Workflow 4: Precision Control

Best for: When you know exactly what you want.

1. Configure all tabs — Settings, Vocal, Structure, Sound
2. Build a custom structure if needed
3. Add specific influences and instrument exclusions
4. Generate with full control

### Workflow 5: Refine an Existing Song

Best for: Improving a Suno output or another AI-generated song.

1. **Analyze** — Paste the existing style prompt or lyrics
2. Review and adjust the auto-configured settings
3. **Lyrics Tab** — Paste the existing lyrics and choose "Rewrite/improve" or "Keep"
4. Generate with your refinements applied
