The right panel is where you interact with your generated song. It contains the action bar at the top and three tabs below.

| Section | Description |
| --- | --- |
| [Action Buttons](/right-panel-action-buttons/) | Generate, export, reset, and open the analyzer |
| [Output Tab](/right-panel-output/) | View and edit generated lyrics, style prompt, and production notes |
| [Chords Tab](/right-panel-chords/) | Chord chart with progression and harmonic notes |
| [History Tab](/right-panel-history/) | Manage, restore, backup, and import saved songs |
