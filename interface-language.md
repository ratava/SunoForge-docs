The SunoForge UI is fully translated into **11 languages**, selectable via the flag dropdown in the top-right corner of the header.

## Switching Language

Click the flag icon in the top-right corner of the header and select your preferred language. All labels, buttons, placeholders, and status messages update instantly. Your preference is saved between sessions.

---

## UI Language vs Song Language

These are two independent settings:

| Setting | Controls |
| --- | --- |
| **UI Language** (flag dropdown) | The language of the SunoForge interface |
| **Song Language** (Settings tab) | The language the AI writes the lyrics in |

You can use the app entirely in German but generate lyrics in Japanese. Changing the UI language does not affect the lyrics language, and vice versa.
